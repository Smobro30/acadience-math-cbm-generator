# Acadience™ Math CBM Generator

A comprehensive web-based worksheet generator for Acadience™ Math Curriculum-Based Measurement (CBM) covering grades K-6.

## Overview

This tool generates customizable math worksheets for all Acadience™ Math skills, including both computation and concepts/applications problems. It provides an intuitive interface for educators to create practice materials tailored to specific grade levels and skill sets.

## Features

- **Grade Coverage**: Kindergarten through Grade 6
- **Skill Types**: 
  - Computation problems
  - Concepts and Applications (word problems)
- **Customization Options**:
  - Adjustable text size (Small, Medium, Large)
  - Variable problem spacing (Compact, Normal, Extra, Extra Extra)
  - Configurable number of columns (2-8 problems per row)
  - Adjustable total problems (10-100)
  - Answer key toggle (show/hide answers)
- **Print-Ready**: Optimized for printing with proper page formatting

## Usage

1. Open [LINK]([url](https://smobro30.github.io/acadience-math-cbm-generator/) in a web browser
2. Select the grade level
3. Choose the skill type (Computation or Concepts/Applications)
4. Select the specific skill to practice
5. Customize worksheet settings as needed
6. Click "Generate Worksheet" to create problems
7. Use "Print Worksheet" to print or save as PDF

## Technical Details

- **Technology**: Pure HTML, CSS, and JavaScript
- **Styling**: Tailwind CSS via CDN
- **No Dependencies**: Runs entirely in the browser, no server required
